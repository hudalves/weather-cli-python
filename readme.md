# Weather CLI ☁️

Projeto simples em Python que consulta previsão do tempo usando a WeatherAPI.

## Funcionalidades

- Buscar previsão do tempo por cidade
- Mostrar temperatura máxima e mínima
- Mostrar chance de chuva
- Previsão de até 3 dias

## Tecnologias usadas

- Python
- Requests
- API REST
